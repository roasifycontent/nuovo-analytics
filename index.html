Connect your GA4

Enter your GA4 property ID and a Google service account key to connect your live data. This is stored locally in your browser — never sent anywhere except Google's API.

Find this in GA4 → Admin → Property Settings. It's the number under "Property ID".
Create a service account in Google Cloud Console → enable Google Analytics Data API → download JSON key → paste here.

Then in GA4 Admin → Property Access Management → add the service account email as Viewer.
