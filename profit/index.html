Settings

Tax (VAT)

Strip VAT from revenue and costs for true profit. The toggle at the top of the page switches between ex-VAT and inc-VAT view.

Revenue ex-VAT uses Shopify's actual tax field per order (accurate for multi-rate / zero-rated). Costs use the assumed rate above. Tick DDP on an order row to exclude that order from VAT stripping (e.g. Alibaba DDP — the supplier handles import VAT).

Recurring costs

Software, apps, AI tools, subscriptions. Pro-rated to the selected date range. Treated like fulfillment for VAT (subject to the "costs entered include VAT" toggle above).

Microsoft Ads spend

Google Ads pulls from GA4 automatically. Microsoft, Meta, TikTok and Other need manual entry.
